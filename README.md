# ProductNexus

A full-stack Inventory Management System built with the MERN stack, featuring a custom monochrome design system, real-time analytics, and a command-line interface for power users.

**Live Demo:** [https://product-nexus-poojan.vercel.app/](https://product-nexus-poojan.vercel.app/)

---

## Tech Stack

**Frontend:**
*   React.js (Vite)
*   Redux Toolkit (State Management)
*   Framer Motion (Animations)
*   Chart.js (Data Visualization)
*   CSS3 (Custom Design System)

**Backend:**
*   Node.js & Express.js
*   MongoDB Atlas (Database)
*   JWT (Authentication)
*   Cloudinary (Image Storage)

---

## Key Features

*   **Global Command Terminal**: Integrated CLI for keyboard-first navigation (`Ctrl+K`).
*   **Real-Time Dashboard**: Live analytics with holographic charts and value tracking.
*   **Secure Authentication**: Full login/registration system with protected routes.
*   **Smart Inventory**: Create, edit, and delete products with optimistic UI updates.
*   **PDF Reporting**: One-click inventory export functionality.
*   **Responsive Design**: Fully optimized for all device sizes.
